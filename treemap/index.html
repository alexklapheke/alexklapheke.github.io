Type the common or scientific name of a tree below and hit enter to add it to the map. For example, Ulmus pumila, Tilia tomentosa, or Amur cork tree.




Cambridge Tree Map

Created by Alexander Klapheke using D3.js. Tree data is from the Cambridge Open Data portal. Basefiles are from Cambridge GIS. Erroneous scientific names and tree diameters are fixed (to the best of my knowledge). The color scheme is from ColorBrewer. This code is released under the MIT license. For more information, see my blog post on the subject.
